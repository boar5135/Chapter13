General Outline for D3

With any given layout in d3, one must first start with a data set. This can either be created in the file or retrieved through a promise. Following the identification of the dataset, one needs to either create an svg element in HTML or through appending it in D3. Next, one might want to use d3 to append a group element to this newly created SVG and provide it with any desired attributes or styling techniques. It is to the group element that the data should be bound. Now, what goes inside this group element is dependent on the type of visualization that one is working to create. If it is the pie layout, one would append paths while a bar chart would append rectangles. In some cases, it may be necessary to bind a subset of data to these elements. This is then followed by a set of attributes that, again, are unique to the task.

Basic Layout of d3

  1. Identify Data Set
  2. Append or create SVG
  3. Append Group Element
  4. Bind Data
  5. Append Task Specific Element to SVG
  6. Apply Any Needed Attributes
